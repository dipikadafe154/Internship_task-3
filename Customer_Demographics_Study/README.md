# Customer Demographics Study 📊

## Overview

Customer Demographics Study is a Python-based data analysis project that helps analyze customer information such as age, gender, and city. The project provides insights into customer distribution and demographic patterns through statistical analysis and visualizations.

This project helps beginners understand data analysis, data visualization, CSV file handling, and Python libraries such as Pandas and Matplotlib.

## Features

* 👥 Analyze customer demographic data
* 📊 Gender distribution analysis
* 🏙️ City-wise customer analysis
* 📈 Average age calculation
* 📉 Bar chart visualization
* 🥧 Pie chart visualization
* ⚡ Beginner-friendly Python project

## Technologies Used

* Python 3
* Pandas
* Matplotlib

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder in VS Code.
4. Open Terminal.
5. Install required libraries:

```bash
pip install pandas matplotlib
```

6. Run the project:

```bash
python demographics_analysis.py
```

## Dataset Columns

| Column     | Description        |
| ---------- | ------------------ |
| CustomerID | Unique customer ID |
| Age        | Customer age       |
| Gender     | Male/Female        |
| City       | Customer city      |

## How It Works

1. Read customer data from a CSV file.
2. Analyze gender distribution.
3. Calculate average customer age.
4. Analyze customer distribution by city.
5. Generate charts and visual reports.
6. Display results in the terminal.
